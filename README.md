# Nikita Putri Prabowo — Personal Portfolio

A clean and responsive personal portfolio website built to showcase my background, projects, work experience, technical skills, and leadership activities as a Computer Science student.

This portfolio highlights my interests and experience in software engineering, web development, data science, machine learning, project management, and quality assurance.

## Overview

This website serves as a centralized portfolio for presenting my academic and professional journey, including selected projects, internship experiences, technical competencies, and organizational involvement.

The design focuses on simplicity, readability, and accessibility, making it easy for recruiters, collaborators, and visitors to explore my work.

## Features

* Responsive personal portfolio website
* Clean and minimal user interface
* Profile introduction with contact information
* Selected project showcase
* Work experience section
* Skills and technology stack overview
* Education and leadership highlights
* Direct links to project demos, GitHub repositories, LinkedIn, and CV
* Ready to deploy on Vercel

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/nikita-portfolio.git
```

Go to the project directory:

```bash
cd nikita-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the website in your browser:

```txt
http://localhost:3000
```

## Editing Portfolio Content

Most of the portfolio content can be edited from:

```txt
src/data/portfolio.ts
```

You can update:

* Personal information
* Project list
* Experience
* Skills
* Education
* Leadership activities
* Contact links

To update the profile photo, replace:

```txt
public/nikita-profile.jpg
```

To update the CV file, replace:

```txt
public/Nikita-Putri-Prabowo-CV.pdf
```

## Deployment

This portfolio is ready to be deployed on Vercel.

### Deploy Steps

1. Push this project to a GitHub repository.
2. Open Vercel.
3. Click **Add New Project**.
4. Import the GitHub repository.
5. Keep the default Next.js settings.
6. Click **Deploy**.
