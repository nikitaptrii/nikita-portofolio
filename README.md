# Nikita Putri Prabowo — Portfolio

A simple personal portfolio website built with Next.js and TypeScript, ready to deploy on Vercel.

## Tech Stack

- Next.js App Router
- TypeScript
- Plain CSS
- Vercel-ready configuration

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit Content

Most portfolio content is in:

```txt
src/data/portfolio.ts
```

Profile image:

```txt
public/nikita-profile.jpg
```

CV file:

```txt
public/Nikita-Putri-Prabowo-CV.pdf
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Open Vercel.
3. Choose **Add New Project**.
4. Import the GitHub repository.
5. Click **Deploy**.
